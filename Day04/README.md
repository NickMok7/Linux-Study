# Day04 - 파일 보기와 파이프

## 배운 명령어

### 파일 내용 보기
- cat : 파일 내용 출력
- head : 앞부분 출력
- tail : 뒷부분 출력
- less : 긴 파일 보기

### 정렬 및 개수
- sort : 정렬
- wc -l : 줄 개수 확인

### 파이프
- | : 앞 명령의 결과를 뒤 명령으로 전달

예시
cat fruits.txt | sort | tail -2

### 리다이렉션
- > : 덮어쓰기
- >> : 이어쓰기

### 기타
- history : 명령 기록
- which : 명령어 위치
